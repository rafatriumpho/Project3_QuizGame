# Quiz Game (Python)

This is my third Python project, built during Week 3 of learning Python.

The program asks the user questions about the Old Testament and keeps track of their score.

## How the Game Works

- The program stores multiple questions and answers  
- The user is asked each question one by one  
- The user types an answer in the terminal  
- The program checks if the answer is correct  
- At the end, the total score is displayed  

## Features

- Multiple questions  
- Answer validation (case-insensitive)  
- Score tracking  
- Immediate feedback (correct / wrong)  

## What I Learned

- How to use lists to store multiple items  
- How to use dictionaries to organize data (question + answer)  
- How to use loops to go through data  
- How to handle user input  
- How to debug errors and fix problems  

## How to Run

Make sure you have Python installed, then run:

python Project3_QuizGame.py
